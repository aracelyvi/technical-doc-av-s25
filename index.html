Introduction

Pikmin Bloom transforms daily steps into growth progress for virtual Pikmin companions. Users walk, collect items, grow Pikmin, and interact with a shared AR world. The app syncs with pedometer APIs and renders real-world overlays using Unity and WebAR.

User Authentication

The app supports OAuth 2.0 for authenticating users through Google or Apple accounts. A session token is issued and refreshed in the background every 50 minutes. Multi-device sync is supported via user-linked cloud accounts.

Step Tracking

Steps are pulled from:

  • Apple HealthKit (iOS)
  • Google Fit (Android)

The data is normalized to account for device-specific variations and synced every 15 minutes. Delta steps are calculated server-side to maintain integrity.

Growth

Each Pikmin begins as a seedling. The seedling grows based on the number of steps walked. The growth formula includes modifiers for flower types, potion boosts, and time of day. A background job computes which Pikmin are ready to pluck every 10 minutes.

Expeditions

Pikmin can be sent on expeditions to retrieve fruits, post cards, and other items.

Expeditions:

  • Are mapped via real-world POIs (Points of Interest)
  • Have variable durations depending on Pikmin strength and number
  • Return data via batched JSON responses

AR Rendering

The AR experience is rendered via a lightweight Unity engine integration, with WebXR fallbacks. The AR module initializes once GPS lock is established.

AR Session Initialization InitializeARSession(gpsData) { loadPikminModels(); placeModelOnGroundPlane(); startTrackingUserMovement(); }
Flower Planting

Users plant virtual flowers by walking in real life.
The system includes:

  • Petal consumption mechanics
  • Flower zone detection
  • Real-time user overlap tracking
  • Time-based bloom cycles
  • Reward distribution at flower density milestones

Daily Journal and Progress

Each day, the app logs user progress, creates a visual journal entry, and syncs it to the cloud.

// Daily journal entry createDailyLog(userSteps, photosTaken) { entry = { date: getToday(), steps: userSteps, photos: photosTaken, events: summarizeDayEvents() }; uploadToJournal(entry); }
App Theming and Skinning

The app supports seasonal skins and event-based themes, including:

  • Halloween Pikmin UI
  • Cherry Blossom Festival background
  • Winter wonderland overlays
  • Regional theming (Japan, USA)
  • Dynamic themes based on time of day
